# Zapret Auto-Selector for Windows

Готовое Windows-приложение с автоматическим подбором стратегий обхода блокировок из проекта **zapret**.

## 🚀 Возможности

- **Автоматический подбор стратегий** из zapret для вашего соединения
- **Мониторинг доступности** популярных сервисов:
  - YouTube
  - Discord
  - ChatGPT
  - Google
  - Cloudflare
  - и другие
- **Автопереключение при деградации** (проверка каждые 5 минут)
- **Журнал событий** с понятными сообщениями на русском языке
- **Сохранение настроек** между запусками
- **Быстрый многопоточный blockcheck** для тестирования доменов/IP
- **GUI на PyQt6** с интуитивно понятным интерфейсом

## 📋 Domain/IP Block Check

Встроенный инструмент для тестирования доступности доменов или IP-адресов с использованием различных стратегий zapret.

## 📦 Установка

### Требования

- Windows 10/11
- Python 3.8 или выше
- [Zapret](https://github.com/bol-van/zapret) (уже включён в папку `bin/`)

### Установка зависимостей

```bash
pip install PyQt6 requests ping3
```

### Запуск приложения

```bash
python main.py
```

Или используйте готовый `.exe` файл (если собран):

```bash
ZapretAutoSelector.exe
```

## 🎯 Использование

### Главный экран

При запуске приложение автоматически:
1. Проверяет доступность целевых сервисов
2. Подбирает оптимальную стратегию zapret
3. Отображает статус каждого сервиса

### Автоматический подбор стратегий

1. Перейдите на вкладку **"Автоподбор"**
2. Нажмите **"Запустить подбор"**
3. Приложение протестирует доступные стратегии и выберет лучшую
4. Результат будет сохранён в настройках

### Мониторинг

- Приложение проверяет доступность сервисов каждые 5 минут
- При обнаружении проблем автоматически переключается на рабочую стратегию
- Все события записываются в журнал

### Журнал событий

Все действия приложения логируются во вкладке **"Журнал"**:
- ✅ Успешные подключения
- ⚠️ Проблемы с доступностью
- 🔄 Переключения стратегий
- ❌ Ошибки

### Настройки

Во вкладке **"Настройки"** можно:
- Выбрать целевые сервисы для мониторинга
- Настроить интервал проверки (по умолчанию 5 мин)
- Включить/выключить автопереключение
- Экспортировать/импортировать конфигурацию

## 🔧 Стратегии zapret

Приложение поддерживает следующие типы стратегий:

- **Fake TLS** - подмена TLS handshake
- **Fake QUIC** - подмена QUIC протокола
- **Host Spoofing** - подмена заголовка Host
- **Custom** - пользовательские стратегии из файлов `.bat`

## 🐛 Решение проблем

### Приложение не запускается
- Убедитесь, что установлен Python 3.8+
- Проверьте установку зависимостей: `pip install PyQt6 requests ping3`

### Стратегии не работают
- Запустите приложение от имени администратора
- Проверьте, что антивирус не блокирует WinDivert

### Сервисы не определяются
- Проверьте подключение к интернету
- Отключите VPN/прокси если используется

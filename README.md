# Zapret Auto-Selector v3.0

Приложение для автоматического подбора и тестирования **существующих стратегий zapret** (`.bat` файлов) с мониторингом сервисов.

## 🚀 Возможности

### Основное
- **✅ Сканирование существующих стратегий** - автоматическое обнаружение всех `.bat` файлов
- **✅ Определение типа стратегии** - Fake TLS, Fake QUIC, MultiSplit, custom
- **✅ Список стратегий** - выбор из всех найденных стратегий
- **✅ Мониторинг доступности** - проверка YouTube, Discord, ChatGPT, Google, Cloudflare
- **✅ Добавление своих сервисов** - через GUI (название + URL)
- **✅ Автопереключение при деградации** - автоматическая смена стратегии при проблемах
- **✅ Компактный интерфейс** - 4 вкладки: Главная, Block Check, Журнал, Настройки
- **✅ Журнал событий** - понятные сообщения на русском языке
- **✅ Сохранение настроек** - конфигурация в `config.json`

### 🔍 Автоподбор стратегий
- **Режим "Все стратегии"** - тестирование всех найденных стратегий
- **Фильтрация по типу** - выбор конкретного типа (Fake TLS, Fake QUIC и т.д.)
- **Кнопка "Тестировать стратегии"** - проверка работы каждой стратегии на сервисах
- **Кнопка "Запустить автоподбор"** - автоматический поиск лучшей стратегии
- **Применение лучшей стратегии** одним кликом

### 📋 Block Check
- **📋 Domain/IP тестирование** - проверка доменов или IP адресов
- **⚡ Многопоточность** - быстрая проверка несколькими потоками (1-16)
- **🎯 Фильтрация по типу** - выбор стратегий по типу (Fake TLS, Fake QUIC, MultiSplit)

## 🔧 Установка

### Требования
- Windows 10/11
- Python 3.8+
- Права администратора (для работы zapret)

### Шаги установки

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Запустите приложение:**
```bash
python main.py
```

3. **(Опционально) Соберите .exe файл:**
```bash
python build.py
```

## 📖 Использование

### Вкладка "🏠 Главная"

#### Статус сервисов
Таблица с текущим состоянием monitored сервисов:
- ✅ OK - сервис доступен
- ❌ FAIL - сервис недоступен
- Время последней проверки

#### Доступные стратегии
Список всех найденных `.bat` файлов с стратегиями:
- Автоматическое определение типа стратегии (Fake TLS, Fake QUIC, MultiSplit)
- Кнопка **"▶️ Запустить"** - запуск выбранной стратегии
- Кнопка **"🎯 Автоподбор"** - автоматическое тестирование всех стратегий

#### Активная стратегия
- Отображение текущей активной стратегии
- Кнопка **"💾 Применить"** - применение выбранной стратегии

### Вкладка "📋 Block Check"

Компактная проверка доступности доменов/IP:

1. **Домен/IP** - введите целевой домен или IP (youtube.com, 1.1.1.1, etc.)
2. **Режим** - выберите тип стратегий для проверки:
   - Все стратегии
   - Fake TLS
   - Fake QUIC
   - MultiSplit
3. **Потоков** - количество потоков для проверки (1-16)
4. Нажмите **"🔍 Запустить проверку"**

Результаты отобразятся в поле ниже.

### Вкладка "📝 Журнал"

- Просмотр всех событий приложения
- Кнопка **"🗑️ Очистить"** - очистка журнала
- Кнопка **"💾 Экспорт"** - сохранение журнала в файл

### Вкладка "⚙️ Настройки"

- **Интервал (мин)** - частота проверки сервисов (1-60 мин)
- **Автопереключение** - включение/выключение автопереключения стратегий
- **Сервисы** - выбор monitored сервисов (Discord, YouTube, etc.)
- Кнопка **"💾 Сохранить"** - сохранение настроек

## ⚙️ Конфигурация

Файл `config.json`:

```json
{
  "active_strategy": "FAKE TLS AUTO",
  "interval": 5,
  "auto_switch": true,
  "services": ["Discord", "YouTube", "Google", "Cloudflare", "ChatGPT"]
}
```

## 🔍 Как работает сканирование стратегий

Приложение автоматически сканирует все `.bat` файлы в директории запуска:

1. Исключается `service.bat` (служебный файл)
2. Из имени файла извлекается название стратегии
3. Анализируется содержимое файла для определения типа:
   - **Fake TLS** - содержит "fake" и "tls"
   - **Fake QUIC** - содержит "fake" и "quic"
   - **MultiSplit** - содержит "multisplit"
   - **custom** - остальные стратегии

## ⚠️ Важные замечания

1. **Права администратора** - для работы zapret требуются права администратора
2. **WinDivert** - убедитесь, что драйвер WinDivert установлен
3. **Антивирус** - может блокировать работу zapret, добавьте исключения

## 🆘 Решение проблем

### Приложение не видит стратегии
- Убедитесь, что `.bat` файлы находятся в той же директории, что и `main.py`
- Проверьте, что файлы имеют расширение `.bat`

### Сервисы показывают "FAIL"
- Проверьте интернет-соединение
- Возможно, требуется другая стратегия zapret
- Используйте Block Check для тестирования

### Ошибки при запуске
- Установите все зависимости из `requirements.txt`
- Запустите от имени администратора
